# Customer_behavior_Analysis
This project demonstrates an end-to-end Data Analytics workflow using Python, SQL (PostgreSQL), and Power BI.
The goal is to extract insights from raw data through data loading, cleaning, exploratory analysis (EDA), SQL querying, visualization, and reporting.

It showcases practical skills required for data analyst roles, including:

Working with datasets in Python

Performing EDA & data preprocessing

Writing SQL queries in PostgreSQL

Building interactive Power BI dashboards

Creating business-ready reports and presentations
